About the Game

Softframe is a perspective-based puzzle experience where players manipulate layered visual frames to restore hidden compositions. Each level gets trickier by adding more layers and making the frames even more out of whack, so you really have to think about how everything fits together in 3D space.

Core Mechanics

Progression introduces multi-layer overlaps and moving frame elements.

  • Drag frames to align visual fragments

  • Adjust scale and rotation within set limits

  • Layer frames in correct depth order

  • Reveal the completed composition

Features

The main thing is keeping everything clear and precise, but not overwhelming - we want players to feel relaxed while they're figuring things out.

  • Perspective-based puzzle system

  • Minimal interface design

  • Layered frame alignment mechanics

  • Abstract and soft color compositions

  • Increasing visual complexity

  • Replayable levels for perfect alignment

How to Play

Players should be able to just dive in without a steep learning curve, and it should feel more like a zen moment than a frantic puzzle rush.
